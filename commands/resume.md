# Command: resume

Minimal session re-entry. Skips the full `go` bootstrap — no auto-update check, no context/stack interview, no gap scanning. Just the latest session knowledge and the next move.

## When to run
- Mid-day re-entry into a session you never really left (context window reset, new terminal tab)
- You already ran `go` once today and just need a fast reminder of where things stand
- Anywhere `go`'s full checklist is overkill

## Dispatches to
`workflows/resume.md`

## Variants
- `/shmorch resume` — no arguments
