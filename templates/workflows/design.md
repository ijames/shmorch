# Workflow: Design
1. Spawn architect with approved spec + relevant analysis
2. Writes shmorch/state/design-<feature>.md
3. Resolve all [DECISION NEEDED] with user — log each to `shmorch/state/decisions.md` **as it's resolved**, not after approval
4. Iterate until approved. If iteration reverses an earlier decision, update that decisions.md entry to reflect current reality — don't leave the old version as a correction footnote.
5. Proceed to Build
